# Site images

Add final photography and graphics here (e.g. WebP or JPEG), then point `src/config/site-images.ts` at the public path.

Suggested filenames (optional):

- `home-hero.webp` — homepage hero (portrait-leaning crop works well)
- `patrick-portrait.webp` — about / home preview
- `weddings-support.webp` — weddings page supporting image (wide)

Replace `public/og-default.svg` with a **1200×630** JPG or PNG for best social preview consistency when you are ready.
